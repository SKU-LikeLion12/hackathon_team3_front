# `끄적임` 고민 상담 커뮤니티 및 마음 치유 서비스
🔗https://team3.sku-sku.com/
 
### 🔗우리의 웹은


- **고민이 있는 사람들이 부담없이 고민을 끄적이며, 다른 이용자들이 건네는 위로와 전문의의 댓글을 통해 간략한 상담을 받을 수 있다.**
-  **더불어 정신적인 질병은 자신이 가지고 있다는 것을 쉽게 인지하기 어려우며, 정신병원에 대한 편견으로 쉽게 방문하기가 꺼려지는 경우들이 발생한다. 이에 따라, 스스로 내면의 아픔을 들여다볼 수 있도록 자가진단 서비스를 제공하여 우울/스트레스/불안 세 분야에 대해 검진을 할 수 있다.**
- **본인 집 주변의 병원을 쉽게 찾을 수 있도록 검색 기능을 통해 보다 빨리 마음의 아픔을 치유할 수 있도록 도움을 제공하고자 한다.**
